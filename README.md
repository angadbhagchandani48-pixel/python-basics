# Python Basics Learning Repository

Welcome to my Python learning journey! 🐍 This repository contains all my Python practice programs, organized by topic.

This repository is organized into logical learning sections to help track progress:

```
python-basics/
├── 01_Basics/              # Basic syntax and first programs
```

## 🚀 How to Use This Repository

1. **Clone the repository:**
   ```bash
   git clone https://github.com/angadbhagchandani48-pixel/python-basics.git
   cd python-basics
   ```

2. **Explore programs:**
   - Navigate to the folder of your interest
   - Read the code and comments
   - Run programs with Python 3

3. **Run a program:**
   ```bash
   python3 01_Basics/first_program.py
   ```

## 💻 Requirements

- Python 3.6 or higher
- No external dependencies (using only Python standard library)

## 📝 Learning Path

Recommended order to go through topics:
1. Start with **01_Basics** for syntax fundamentals
2. Move to **02_DataTypes** to understand data types
3. Learn **03_Operators** to perform operations
4. Master **04_ControlFlow** for decision-making
5. Practice **05_Loops** for repetition
6. Explore **06_DataStructures** for data organization
7. Apply everything in **07_ProblemSolving**
